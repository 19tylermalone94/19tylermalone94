---

Full-stack engineer with a CS degree from CSU and a background that goes: Short-order cook → Marine Corps → school → shipping real software. I've spent the last year deep in AI platform work — backends, pipelines, real-time systems, infra, the whole thing. I'm comfortable owning a system end-to-end and equally comfortable dropping into an existing codebase and getting up to speed.

Most recently I've been building [WarTable](https://wartable.ai). I can't say much, but it's been a serious technical lift and I've learned a lot.

Relocating to Japan. Open to remote roles or teams based there.

---

## what I can do

**Backend & systems** — production APIs, real-time event streaming, task queues, auth, database design and schema evolution, testing and CI/CD

**AI & agents** — RAG pipelines, agentic workflows with tool calling, LLM orchestration, third-party integrations via MCP, conversational and workspace memory systems

**Frontend** — Next.js, React, real-time UI, component systems, the stuff that has to actually work when users touch it

**Infrastructure** — Docker, GCP, GitHub Actions, staging/prod isolation, automated deployments and migrations. I set things up so they don't need babysitting
